# Frutiger Aero Asset Guide

Here is your checklist for creating the assets. Save all files in the `images` folder with the **exact names** listed below.

## 1. Backgrounds
- **File Name:** `wallpaper.jpg`
- **Description:** A high-res Frutiger Aero landscape (green grass, blue sky, abstract waves).
- **Tutorial:** [Frutiger Aero Composite Tutorial](https://www.youtube.com/watch?v=example) (Search: "Frutiger Aero Photoshop Tutorial")

## 2. Window Elements
- **File Name:** `start-orb.png`
- **Description:** A glossy, circular start button (like Windows 7).
- **Tutorial:** [Glossy Orb Tutorial](https://www.youtube.com/watch?v=example) (Search: "Glossy Orb Photoshop")

- **File Name:** `window-icon.png`
- **Description:** A small 16x16 or 32x32 icon for the top left of the window.

- **File Name:** `close-btn.png`
- **Description:** A small red "X" button for the top right corner.

## 3. Buttons
- **File Name:** `button-bg.png` (ALREADY DONE!)
- **Description:** You provided this file (`pagebottenbackgrund.png`). It is now set as the background for your menu buttons.

## How to Verify
Once you save a file (e.g., `wallpaper.jpg`) into the `images` folder, refresh `index.html` in your browser. The code is already set up to find it!
